# 💸 Money Currency Tracker

A simple, visually appealing web app to help you track and convert currencies. Perfect for budgeting, travel, or keeping an eye on your favorite exchange rates!

---

## ✨ Features
- Clean, modern, and responsive design
- Easy-to-use interface for entering and converting amounts
- Supports multiple currencies (manual input)
- No account or login required
- **No external API required** (yet!)

---

## 🚧 API Notice
> **Note:** This project currently does **not** use any external currency API. All rates and conversions must be entered manually. API integration is planned for a future update!
